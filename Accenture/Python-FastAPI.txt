def two_sum(nums, target):
    d = {}
    for i, n in enumerate(nums):
        if target - n in d:
            return [d[target-n], i]
        d[n] = i

2Ô∏è‚É£ Sliding Window Rate Limiter
```python
import time
from collections import deque

class RateLimiter:
    def __init__(self, limit, window):
        self.limit = limit
        self.window = window
        self.q = deque()

    def allow(self):
        now = time.time()
        while self.q and self.q[0] < now - self.window:
            self.q.popleft()
        if len(self.q) < self.limit:
            self.q.append(now)
            return True
        return False
```

3Ô∏è‚É£ Debounce Function
```python
import time

def debounce(wait):
    def decorator(fn):
        last = 0
        def wrapper(*args):
            nonlocal last
            now = time.time()
            if now - last > wait:
                last = now
                return fn(*args)
        return wrapper
    return decorator
```

1Ô∏è‚É£ ASGI vs WSGI
WSGI = sync only, ASGI = async + WebSockets + long-lived connections.
FastAPI uses ASGI, so it supports real-time apps.

2Ô∏è‚É£ Uvicorn vs Gunicorn
Uvicorn ‚Üí ASGI server (async, fast, dev-friendly)
Gunicorn ‚Üí Process manager (production, multi-workers)
üëâ Best practice: Gunicorn + Uvicorn workers in prod.

3Ô∏è‚É£ Pydantic v1 vs v2
v1: Pure Python, slower validation
v2: Rust-powered core (pydantic-core), 5‚Äì50x faster, new validators
üëâ FastAPI latest versions prefer Pydantic v2

make function async
